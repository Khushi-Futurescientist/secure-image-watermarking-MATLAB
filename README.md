# Secure Image Watermarking using MATLAB

## Overview
This project implements a secure digital image watermarking system for academic document authentication using Least Significant Bit (LSB) embedding and key-based XOR encryption.

The watermark is generated from a registration number and embedded into an image using pseudo-random embedding positions to improve security and resistance against unauthorized extraction.

---

## Features
- LSB-based watermark embedding
- XOR-based watermark encryption
- Pseudo-random embedding position selection
- Watermark extraction and authentication
- Robustness evaluation under multiple attacks

---

## Technologies Used
- MATLAB
- Digital Image Processing
- Image Watermarking
- Encryption Techniques

---

## Methodology
1. Watermark generation from registration number
2. XOR encryption using key-based security
3. Pseudo-random embedding position generation
4. LSB embedding into grayscale image
5. Watermark extraction and verification

---

## Performance Evaluation Metrics
The system performance was evaluated using:
- PSNR (Peak Signal-to-Noise Ratio)
- SSIM (Structural Similarity Index)
- BER (Bit Error Rate)
- Correlation Coefficient

---

## Attack Analysis
The robustness of the watermarking system was tested under:
- Gaussian Noise
- Salt & Pepper Noise
- Blur Attacks
